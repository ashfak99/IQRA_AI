# IQRA AI

# 📖 Quran Semantic Search Engine with RAG

A **Retrieval-Augmented Generation (RAG)** chatbot that answers user queries based on the Holy Quran. It performs **semantic search** over Quranic verses (Arabic + translations) and generates accurate, context‑aware responses with proper Surah & Ayah citations.

## 🚀 Features

- 🔍 **Semantic Search** – Understands meaning, not just keywords. Works for Hinglish, English, Urdu, and Arabic queries.
- 📚 **Quranic Knowledge** – Uses authentic Quran text with translations (English, Hindi, Urdu).
- 🧠 **RAG Pipeline** – Retrieves relevant verses and generates answers strictly from the retrieved context.
- 📌 **Citations** – Every answer includes Surah name and Ayah number(s).
- 🛡️ **Hallucination Control** – Returns `"I don't know"` when no relevant verse is found.
- 🌐 **Multilingual** – Embedding model supports 100+ languages.
